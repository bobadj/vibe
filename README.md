# Vibe

### Requirements
* [node.js](https://nodejs.org/en) >= 20
* [pnpm](https://pnpm.io/) >= 8

### Development setup

* clone repository
```bash
git clone git@github.com:bobadj/vibe.git
```
* install dependencies with `pnpm`
```bash
cd vibe
pnpm install 
```
* start development server
```bash
pnpm run dev 
```
or
```bash
npm run dev
```
